# Contributing to Portfolio App

Thank you for your interest! This project is primarily maintained for the CVHowlader team, but we welcome external issue reports and pull requests.

## How to Contribute

1. **Fork the repository**
2. **Create a topic branch** for your changes
3. **Make your contribution**
4. **Ensure all tests pass** (if relevant)
5. **Open a Pull Request** describing your changes

## Reporting Issues

You’re welcome to [open an issue](https://github.com/Charlesbasis/portfolio-app/issues/new). Please include:
- A clear title and description
- Steps to reproduce (if relevant)
- Screenshots or logs if you have them

## Community

Please see our [Code of Conduct](CODE_OF_CONDUCT.md).

Thank you for improving Portfolio App!
